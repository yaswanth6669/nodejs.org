# 诊断指南

这些指南在[诊断工作小组](https://github.com/nodejs/diagnostics)中得以创建，
其目的旨在诊断一个程序中的相关问题时能给予相应的指导。
本文档基于用户亲历组织而成。这些历程是一系列互相关联、一步步的措施。
用户应当遵从他们，以便于根据用户上报的事件锁定问题。
