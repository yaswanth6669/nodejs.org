# 使用内存检查器

在本地环境中，当我们谈到“在线调试”时，这意味着把我们的程序附加到调试器上，然后在代码里加
若干断点从而挂起正常的程序执行。然后我们单步调试完整个代码路径，观察不同单步时堆的变化。
不过在生产环境中使用调试器于我们而言并不是一个可选项，因为我们限制了该机器的访问权限，同时
我们也无法打断正常的工作执行流。道理很简单：因为它正在处理与公司业务逻辑相关的重要任务。
